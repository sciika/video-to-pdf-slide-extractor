# video-to-pdf-slide-extractor
講義やセミナーの動画ファイル（.mp4）からスライドが切り替わった瞬間を自動で検出し、不要なフレームを削除した上で、最終的に1枚のPDF資料に自動結合するPythonスクリプトです。

## 💡 開発の背景・目的
オンライン講義の復習の際、動画を何度も見返す非効率さを解決するために開発しました。
動画内のスライド変化を自動検知して電子資料化することで、学習効率の向上を目的としています。

## ✨ 主な機能と工夫した点
- **OpenCVを用いた変化検知**: フレーム間の差分の絶対値（`cv2.absdiff`）の平均値を算出。設定した閾値を超える変化があった瞬間の「スライドが切り替わったフレーム」のみを正確に抽出します。
- **ヒューマン・イン・ザ・ループ（手動チェック機能）**: AIやプログラムの誤検知（講師の顔アップなど）をパージするため、PDF結合の手前でエクスプローラーから不要な画像を目視で手動削除できる一時停止機能を組み込みました。
- **クリーンアップ処理**: PDF生成後、作業用に抽出した大量の画像フォルダを自動で削除するクリーンアップ設定（`KEEP_WORK_IMAGES`）を実装し、PCのストレージを圧迫しない設計にしています。

## 🛠️ 技術スタック
- Python 3.x
- OpenCV (`opencv-python`)
- NumPy
- img2pdf
